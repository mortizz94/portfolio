Bienvenido al repositorio de mi portafolio personal y profesional. Este proyecto refleja mi trayectoria como **Senior Operations Lead**, mostrando mis habilidades técnicas, experiencia en liderazgo y proyectos de desarrollo.

🌐 **Demo:** [glutendo.com](https://glutendo.com) (Enlace placeholder/ejemplo)

## ✨ Características

- **Diseño Premium & Moderno:** Estética "Cyber/Tech" con paleta oscura, gradientes eléctricos y efectos de cristal (Glassmorphism).
- **Animaciones Suaves:** Transiciones fluidas y efectos de aparición al hacer scroll (scroll-reveal).
- **Totalmente Responsivo:** Adaptado perfectamente a móviles, tablets y escritorio.
- **Sin Dependencias Pesadas:** Construido con **Vanilla HTML, CSS y JS** para máximo rendimiento y velocidad de carga.
- **Secciones:**
  - **Hero:** Presentación impactante con estado de disponibilidad "En línea".
  - **Trayectoria:** Línea de tiempo interactiva de mi carrera profesional.
  - **Stack Técnico:** Visualización de habilidades en Infraestructura, Cloud y Dev.
  - **Formación:** Detalles académicos y certificaciones.
  - **Contacto:** Footer moderno con enlaces directos a Email, GitHub y LinkedIn.

## 🛠️ Tecnologías

Este proyecto apuesta por la simplicidad y el rendimiento, evitando frameworks pesados innecesarios para una web estática de este tipo.

- **HTML5:** Semántico y accesible.
- **CSS3:** Variables CSS (Custom Properties), Flexbox, CSS Grid y animaciones keyframes.
- **JavaScript (ES6+):** Lógica ligera para el menú de navegación, botón "Volver arriba" y observadores de interacción.
- **Font Awesome:** Iconografía vectorial.
- **Fuentes:** Inter, JetBrains Mono y Manrope (Google Fonts).

## 🚀 Instalación y Uso Local

Para ver este proyecto en tu máquina local:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/mortizz94/portfolio.git
    ```
2.  **Navegar al directorio:**
    ```bash
    cd portfolio
    ```
3.  **Abrir `index.html`:**
    Simplemente abre el archivo `index.html` en tu navegador web favorito (Chrome, Firefox, Edge, etc.).
    *Opción Recomendada:* Usa una extensión como "Live Server" en VS Code para simular un servidor local.
